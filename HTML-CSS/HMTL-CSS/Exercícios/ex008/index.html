Trabalhando com listas

Listas ordenadas

Usando a tag <ol> é possível criar uma lista ordenada e adicionar os elementos entre a tag <li>. Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3

Listas não ordenadas

Feitas a partir da tag <ul>. Exemplo:

  • Acordar
  • Tomar café
  • Malhar

Juntando listas

  1. Malhar
    • Corrida com joelho alto
    • Escalador alternado
  2. Tomar banho
  3. Arrumar

Lista de definições

Feitas com a tag <dl>. Exemplo:

PlayStation
Melhor console da atualidade
Xbox
Pior console de todos os tempos
