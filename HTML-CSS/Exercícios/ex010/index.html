Imagem dinâmica

Usando a tag <picture> é possível adicionar imagens ao site.

Com o auxílio da tag <source> é possível selecionar o tamanho máximo/mínimo em pixels em que ocorrerá a transição de imagens para que ela se ajuste ao tamanho da tela sem aparecer a barra de rolagem lateral.

Reproduzindo áudio

Para reproduzir um áudio basta usar a tag <audio> com o parâmetro "autoplay", além do parâmetro "controls" para que o usuário possa pausar o áudio.

Para que o áudio fique melhor configurado, pode-se utilizar a tag <source>.
