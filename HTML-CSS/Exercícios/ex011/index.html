Inserindo vídeos hospedados localmente

Para adicionar um vídeo basta usar a tag <video>, que também pode conter a tag <source> para configurar melhor a exibição do arquivo.

O vídeo a seguir está hospedado no meu próprio servidor.

Inserindo vídeos hospedados externamente

Para adicionar um vídeo do YouTube, clique em "compartilhar" no vídeo e clique em "incorporar" para obter o código em HTML.
